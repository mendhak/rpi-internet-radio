Raspberry Pi Internet Radio
==================

My own implementation of an Internet radio player on a Raspberry Pi with a breadboard. 

##Ingredients

* 3x 10K Ω resistors
* 1x 100 Ω resistor
* 1x LED
* 3x momentary push buttons
* 1x breadboard
